## Project Setup

```sh
npm install

or

yarn install
```

### Compile and Hot-Reload for Development

```sh
npm run dev

or

yarn dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build

or

yarn [run] build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint

or

yarn [run] lint
```
